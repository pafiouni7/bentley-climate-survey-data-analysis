# Perception of University Core Values

## Overview
This project analyzes how demographic and background factors influence perceptions of Bentley University’s core values: caring, collaboration, diversity, honesty, impact, learning, and respect.

## Key Insights
- Learning is the highest-rated value; diversity is the lowest  
- Ratings are strongly correlated across all values  
- Socioeconomic status (SES) is the strongest driver of perception  
- Race and gender are not significant predictors  
- Graduates tend to give higher ratings  

## Methods
- Exploratory Data Analysis (heatmaps, boxplots, correlations)  
- Regression modeling to identify significant predictors  
- Logistic Regression (~60% accuracy) to classify high ratings  
- Random Forest to evaluate feature importance  

## Conclusion
Perceptions of core values are driven more by socioeconomic background than by identity factors.

## Recommendation
Focus on improving experiences for lower SES groups, especially in areas like diversity where ratings are lowest.

## Tech Stack
Python, Pandas, NumPy, Matplotlib, Seaborn, Statsmodels, Scikit-learn
